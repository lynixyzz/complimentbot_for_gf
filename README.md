# complimentbot_for_gf 🐱

Бот-компаньон для коти: комплименты и поддержка по настроению, предупреждения о
плохой погоде и веб-апп «фраза дня» с космической сферой.

## Структура

| Файл | Что делает |
|------|-----------|
| `bot.py` | точка входа: собирает всё и запускает polling |
| `config.py` | переменные окружения + постоянное состояние (`state.json`) |
| `content.py` | загрузка `data/*.json`, категории кнопок, фраза дня |
| `handlers.py` | хендлеры Telegram: эмоции, админ-панель, блокировка чужих |
| `weather.py` | прогноз Yandex Weather + фоновый поток предупреждений |
| `server.py` | мини-сервер веб-аппа (`/` и `/phrase`) |
| `webapp/index.html` | вращающаяся радужная сфера + фраза дня |
| `data/*.json` | фразы по настроениям, по погоде (`weather.json`), фразы дня (`phrases.json`) |
| `tests/test_bot.py` | проверки чистой логики |

## Запуск

```bash
pip install -r requirements.txt
cp .env.example .env   # заполнить BOT_TOKEN, ALLOWED_USERS, ADMIN_IDS
python bot.py
```

## Переменные окружения

- `BOT_TOKEN` — токен бота (обязателен).
- `ALLOWED_USERS` — id котей через запятую. Любой, кого нет в списке, получает «Отказано! Вы не котя!».
- `ADMIN_IDS` — id админов через запятую. Получают уведомления о настроении коти и доступ к `/admin`.
- `YANDEX_WEATHER_KEY` — ключ Yandex Weather API. Без него погодные предупреждения отключены.
- `WEBAPP_URL` — публичный **https**-адрес веб-аппа. Без него кнопка «Фраза дня» скрыта.
- `WEBAPP_PORT` — порт локального сервера веб-аппа (по умолчанию 8080).

## Админ-панель

`/admin` (только для `ADMIN_IDS`):
- **📍 Задать геолокацию** — прислать точку 📍 или написать «широта,долгота». По ней проверяется погода.
- **🔄 Сбросить фразу дня** — выдать новую фразу дня прямо сейчас.

## Погода

Раз в 3 часа проверяется прогноз на завтра. Если ожидается дождь / гроза / снег и
сегодня ещё не предупреждали — котям уходит случайная цитата из `data/weather.json`
под нужное условие.

## Веб-апп

`server.py` отдаёт `webapp/index.html` и `/phrase`. Чтобы Telegram открыл его,
`WEBAPP_URL` должен указывать на публичный https (продакшн-хостинг или туннель
вроде `cloudflared`/`ngrok`, проксирующий на `WEBAPP_PORT`).
